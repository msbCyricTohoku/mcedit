 :: MINIMALIST TERMINAL EDITOR v1.0 ::


 [ ABOUT ]
   A raw, lightweight, and efficient text editor for the console.
   Inspired by Kilo. Built for speed. No bloat.
   
   Supports syntax highlighting for:
   > C / C++
   > Python
   > Fortran
   > PHITS


 [ INSTALL ]
   $ make
   $ sudo make install


 [ USAGE ]
   $ mcedit <filename>

   +-----------+-----------------------+
   | COMMAND   | ACTION                |
   +-----------+-----------------------+
   | CTRL + S  | Save Buffer           |
   | CTRL + Q  | Quit (Escapes too)    |
   | CTRL + F  | Find Text             |
   | CTRL + G  | Jump to Line          |
   +-----------+-----------------------+
